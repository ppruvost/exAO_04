# Sonomètre Web

Un projet simple pour mesurer le niveau sonore via le navigateur, avec une barre de progression colorée et une alerte sonore en cas de niveau trop élevé.

## Fonctionnalités
- Mesure du niveau sonore en temps réel.
- Barre de progression colorée (vert/orange/rouge).
- Alerte sonore en cas de niveau rouge.

## Prérequis
- Navigateur moderne (Chrome, Firefox, Edge).
- Accès au microphone.

## Installation
1. Clonez ce dépôt.
2. Ouvrez `index.html` dans un navigateur.

## Utilisation
- Cliquez sur "Démarrer" pour activer le sonomètre.
- Cliquez sur "Arrêter" pour désactiver.
